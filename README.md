# CODEX
A browser-based puzzle game about reverse-engineering context-free grammars.

### The Game
You can [play it here](https://ed-von-schleck.github.io/codex/).

You are presented with a series of "Target Strings"â€”sequences of symbols generated by a hidden set of rules. Your goal is to reconstruct those rules.

By dragging and dropping symbols to create logic chains, you must build a grammar that matches the patterns. When hovering over a solved pattern, you can see the derivation steps.

### The Tech
CODEX is built entirely in vanilla JavaScript, utilizing no external frameworks or build tools. It implements the Cocke-Younger-Kasami algorithm to parse user-defined grammars against target strings in real-time.

### How to Run

1.  Clone this repository.
2.  Run `python -m http.server` (or some other static file server) inside the root directory.
